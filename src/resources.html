          Resources

          There is so much inspiring content available. Books, News, Podcasts, Movies, articles... Here is some of it that I enjoyed very much and have been inspired by.

          Book

          Start with why - Simon Sinek

          Awesome book that helps you to think about 'the why' first.

          Book

          Inspired - Marty Cagan

          Amazing book that outlines the basics of product management.

          Book

          User story mapping - Jeff Patton, Peter Economy

          The book about user story mapping.

          Book

          Zero to one - Peter Thiel

          Thiel explanes what it takes to be truly successful in today's markets.

          Book

          The lean startup - Eric Ries

          This book will teach you the process of adding true value.

          Book

          Clean Code - Robert C. Martin

          Great resource about best practices in software development
